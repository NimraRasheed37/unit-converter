                 Project2: Unit Convertor web app

installations:
streamlit
UV


tasks:
-> user will enter a value to convert
-> user will select units (from _____ to _____)
-> user will click on convert button
->application will calculate conversion using a predefined formula
-> Result is displayed

steps: 
1. install uv: ( pip install uv )
2. initializing uv project: ( uv init streamlit )
3. add streamlit in new project folder: ( uv add streamlit )
4. .venv folder will be created than run this command: ( .venv\scripts\activate )
5. import stramlit in code
6. define function
7. add streamlit UI
8. run program in chrome: ( streamlit run unit_converter.py )
